Dependencies
=== 

Given our demo client is a Maven based project, the build tool will download the required binaries for our platform, we just need Java 21.
* Java 21 LTS (minimum)

How to build:
===

```bash
mvn clean install
```

How to run the client
===
```bash
mvn clean javafx:run
```
